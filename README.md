# Data Analyzer and Transformer Program

This is a simple Python menu-based program.

The program allows the user to:

* Enter numbers into a list
* Display data summary (count, min, max, sum, average)
* Calculate factorial using recursion
* Filter data using a lambda function
* Sort data in ascending or descending order
* Display dataset statistics using multiple return values

## Menu Options

1. Input Data
2. Display Data Summary
3. Calculate Factorial
4. Filter Data by Threshold
5. Sort Data
6. Display Dataset Statistics
7. Exit

## Concepts Used

* Lists
* Functions
* Recursion
* Lambda function
* Built-in functions
* match-case statement

## Requirements

* Python 3.10 or above

## How to Run

```
python main.py
```
